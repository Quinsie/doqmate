# DoQ-Mate
---
우리들의 문서 기반 분석 에이전트, 친구 -- 도큐메이트. (DoQ-Mate)

## 배경
---
nbsp;현시대에는 챗봇 형태를 하는 많은 언어모델들이 즐비하며, 많은 이용률을 보인다. 하지만 어떻게든 답변을 생성하여 돌려주는 챗봇의 특성상 환각을 보일 때가 있다. 문제는 이러한 환각을 챗봇 자신도, 이용하는 사람도 환각임을 잘 인지하지 못한다는 점이다. 무서운 점은 사실과 거짓말이 섞여 들어온다는 점이다. 해당 분야에 대해 해박한 지식을 갖고 확인차 질문하는것이 아니라면, 챗봇의 응답을 사실이라고 믿을 수밖에 없게 된다.
nbsp;또한, 이런 챗봇들은 현재 RAG 구조를 채용하고있다. 레퍼런스에 사용할 자료를 넘겨준 뒤, 해당 자료를 기준으로 응답을 생성하는 기능이다. 여기서 문제는, 자료의 양이 방대해질수록 챗봇은 한 번에 전부를 파악하고 답변을 수행하기 힘들어한다. 이 과정에서 자연스레 환각의 비율이 암암리에 높아지는 모습을 보인다.
